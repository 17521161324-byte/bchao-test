## API 清单

### 录音管理
| 方法 | 路径 | 说明 |
|------|------|------|
| GET | /api/audio/tree | 录音文件树 |
| GET | /api/audio/batches | 批次统计 |
| GET | /api/audio/patients?date=xxx | 按日期筛患者 |
| GET | /api/audio/records?date=xxx | 平铺记录 |
| GET | /api/audio/status | 总体统计 |
| GET | /api/audio/verify | 数据核对 |
| DELETE | /api/audio/patient/{id} | 删除患者 |
| POST | /api/audio/scan | 扫描目录 |

### 结果管理
| 方法 | 路径 | 说明 |
|------|------|------|
| POST | /api/result/upload | 上传 xlsx |
| GET | /api/result/{record_id} | 按病历号查 |
| PUT | /api/result/{result_id} | 更新结果（支持卵泡明细）|

### 模型配置
| 方法 | 路径 | 说明 |
|------|------|------|
| GET | /api/model | 列表 |
| POST | /api/model | 创建 |
| PUT | /api/{id} | 更新 |
| DELETE | /api/{id} | 删除 |
| POST | /api/{id}/test | 连通性测试 |
| POST | /api/init-defaults | 初始化默认 |

### 批量实验
| 方法 | 路径 | 说明 |
|------|------|------|
| GET | /api/experiments | 列表 |
| POST | /api/experiments | 创建批次 |
| GET | /api/experiments/{id} | 详情 |
| POST | /api/{id}/combinations | 添加组合 |
| PUT | /api/{id}/combinations/{cid} | 更新组合 |
| DELETE | /api/{id}/combinations/{cid} | 删除组合 |
| PUT | /api/{id}/patients | 更新患者范围 |
| POST | /api/{id}/start | 启动 |
| POST | /api/{id}/pause | 暂停 |
| POST | /api/{id}/resume | 继续 |
| POST | /api/{id}/cancel | 取消 |
| POST | /api/{id}/retry | 重试失败 |
| GET | /api/{id}/progress | 实时进度 |
| GET | /api/{id}/tasks | 任务列表 |
| GET | /api/{id}/metrics | 聚合指标 |
| GET | /api/{id}/results | 结果列表 |

## 数据结构

### 数据库表（当前）
- `date_folders` — 日期批次
- `patient_records` — 患者
- `audio_segs` — 音频分段
- `b_ultra_results` — B超金标准
- `model_configs` — 模型配置
- `test_runs` — 测试记录
- `experiment_batches` — 实验批次
- `experiment_combinations` — 实验组合
- `experiment_tasks` — 实验任务

### 用户数据（当前）
- 4 个日期批次（20260622~20260626）
- 54 个患者
- 406 个音频分段
